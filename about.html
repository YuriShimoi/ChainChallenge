Game Instructions

Hide the game grid when active, and Show when deactive


When active, move the items on grid by the positions clicked


Change the game color theme, currently the Dark Theme is the principal


Open in a new page the configuration code of currently level


When active, show informations about the items clicked on grid


When active, destroy the items clicked on grid


When clicked, show informations about the way to win and lose the game


Some items get things when clicked on grid, try it




About The Game

ChCh (Chain Challenge) is a 1 week project game, developed for fun and non-profit, with the goal of creating a malleable platform to create logistics and automation games with a certain level of standardization, easy to configure and public format for all users and developers.
The features and gameplay have been implemented in a way that is easy to understand and modify, the whole code of this platform is available on Github.

More ready levels will be added depending on the time to demonstrate the different ways to set up a level, all settings can be used as desired, send your configuration via message in Github to me, Yuri Shimoi. ;D
