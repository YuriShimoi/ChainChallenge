ChCh - Make a Configuration

How To

  • ▹ Make at least one of each item
  • ▹ Names must only contain letters and spaces
  • ▹ Images are HTML5 tags, work best as <i>, has support to <img>, doesn't work as simple text, <p> or <span>
      Examples of <i>, look at Font Awesome:
    • <i class="fas fa-city"></i> ⇢
    • <i class="fas fa-hard-hat"></i> ⇢
      • Direct link with Imgur:
    • <img src="https://i.imgur.com/tFixaCw.png"> ⇢
      • Direct link with Postimg:
    • <img src="https://i.postimg.cc/mrz7W9ZW/icon.png"> ⇢
      • Image found in Dreams Time:
    • <img src="https://thumbs.dreamstime.com/b/fresh-chocolate-chip-cookie-isolated-white-above-dark-chocolate-chip-cookie-isolated-white-bakcground-top-view-128510868.jpg"> ⇢
  • ▹ To set more than one material to Price, Gain, Spent or Click, separate names and amounts by ; without spaces and order then respectively
      Example:
    • Name: Iron;Bronze;Gold
    • Amount: 10;100;200
    • Result in: Iron:10, Bronze:100, Gold:200
  • ▹ If your builder does not spend or gain per click, just set the name to null
  • ▹ The chronometer format is 'minutes seconds'
  • ▹ Set Amount in Condition to negative to 'less or equal than' (for zero as well), and positive to 'greater or equal than'
  • ▹ Hover for a short description
